Registers custom post types and custom fields for the ishop theme

== Description ==

**The ishop Toolbox plugin is meant to be used only with ishop themes that require it.**

This plugin registers custom post types and custom fields

== Installation ==

1. Go to Plugins > Add New and search for ishop Toolbox. Install and activate it.
2. That's it. No configuration needed.
